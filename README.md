# Computer Vision Practicals
This repository contains various computer vision practicals and labs.

## Directory Structure
### Practicals
- **P1**: Spatial Transformations: Convolution & Correlation
- **P2**: Image Resizing and Interpolation (Bilinear, Bicubic, Nearest Neighbor, Lanczos)
- **P3**: Image Color Models, Noise Addition and Filtering
- **P4**: Image Processing using Fourier Transform
- **P5**: Wavelet Transform
- **P6**: Edge Detection (Sobel, Prewitt, Roberts)
- **P7**: Watershed Segmentation for Image Segmentation
- **P8**: Active Contours (Snakes) Segmentation
- **P9**: Feature Matching and Triangulation
